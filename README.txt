Zenya Tutorial - build a pixel art paint app

- Build a tile-drawing app
- Build and manage a custom grid
- Use tkinter colour picker using the selected colour to set a brush colour
- Implement paint and erase functionality, call a function when a cell pressed to either paint it or erase it
- Save app content as an image, using the Pillow library to grab a section of the screen

Steps
- Build a 20x10 grid
- Build a control panel
- Implement colour picker
- Draw and erase functionality
- Save the image
